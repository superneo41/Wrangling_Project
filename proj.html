Introduction

1. Problem

Is there a relation between State Household Income, per capita Starbucks & per capita McDonaldâ€™s ? The reason I came up with this question is because I observed that there are more students in Starbucks when it is finals week. People need coffee to stay focus on the study. So I wonder if people in one state consume more Starbucks coffee, does that means these people are more productive and make that state average income higher ? Similarly, what about McDonaldâ€™s ?

2. Data

I obtained the average household income data and population data from the CENSUS using an API key.
I scraped the starbucks stores data from StateMaster Website:
http://www.statemaster.com/graph/lif_sta_sto-lifestyle-starbucks-stores#definition
I download the McDonaldâ€™s stores data from Statista Website:
https://www.statista.com/statistics/631235/number-of-mcdonald-s-us-by-state/

3. Approach

I try to visualize all the data. First I plot some maps showing the household income, number of Starbucks/McDonaldâ€™s to get a rough idea about which states has the high income and which states has the most Starbucks/McDonaldâ€™s stores. Then I fit a simple linear regression to see if income truely related to S/M (Starbucks/McDonaldâ€™s). Finally, I analyse the regression model and get some insights.

Conclusion

  1. State average household income do have relation with the number of Starbucks/McDonaldâ€™s per 1000 people in that state (rate of Starbucks/McDonaldâ€™s). But average income is not related to the number of Starbucks/McDonaldâ€™s in that state
  2. The higher rate of Starbucks the higher average income of the state
  3. The higher rate of McDonaldâ€™s the lower average income of the state
  4. There is pretty good linear relation between Starbucks rate and the average income. We can even roughly estimate the state average income by the starbucks rate
  5. The linear relation between income and starbucks rate doesnâ€™t necessarily mean they are in a causal relationship. There are more worth digging !
